# Отслеживание нутриентов

## Что отслеживается?

Приложение автоматически сохраняет:
- Название блюда
- Вес порции (г)
- Калории (ккал)
- Белки (г)
- Жиры (г)
- Углеводы (г)

## История

Все приёмы пищи отображаются в хронологическом порядке:

1. Утро — Овсянка (320 ккал)
2. Обед — Куриная грудка с рисом (480 ккал)
3. Ужин — Салат с авокадо (290 ккал)

## Анализ

Ежедневно система показывает:
- Выполнение норм по калориям и макронутриентам
- Прогресс по цели (например, «-0.8 кг за неделю»)
